Hostingbedrijf COLICSS is een startende onderneming die goedkope webhosting aanbiedt aan klanten. Het bedrijf wordt geleid door dhr. W. Beltman. Ze hebben altijd een omgeving draaiende gehad om Windows Server maar omdat ze graag Open Source willen werken willen ze overgaan naar Linux. Kort samengevat hebben ze dus een omgeving in windows server en willen ze graag een testomgeving voor linux omdat ze graag in Open Source willen werken.
